## Trust Section

**Heading:** Built for infrastructure trust boundaries

**Subheading:** The strongest product differentiators are also the simplest ones: private credential custody, bring-your-own AI, agent-managed workflows, and explicit approval before anything changes.

### Private by Default
**Headline:** Credentials stay on your machine
**Body:** Run checks against existing cloud accounts and cluster contexts without handing credentials to a hosted SaaS layer.

### BYO AI
**Headline:** Bring your own model keys
**Body:** Use the models you prefer and keep direct cost control instead of paying another token markup layer.

### Agents
**Headline:** Can be managed by your agent, or manage your agent
**Body:** Clanker Cloud can be managed from your own agent workflows, and Clanker Cloud can also manage your agent, so infrastructure context and controlled actions stay inside the tools you already run.

### Read-only First
**Headline:** Review the plan before anything changes
**Body:** Clanker Cloud checks live context first, generates plans second, and only makes changes in explicit maker mode.
